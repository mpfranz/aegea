## Git Commands

1. See a list of remote repositories we are connected to.
    -> git remote -v

2. Check the current status of git.
    -> git status
    
3. Add all files to local git environment.
    -> git add .

4. Commit all files to local repository.
    -> git commit -m "Describe this commit"

5. Push local repository to the remote repository.
    -> git push origin master
